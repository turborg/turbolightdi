## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please **do not** open a public issue. Instead, follow these steps:

1. **Email us:** Send a detailed report to [ security@turborg.com ].
2. **Details to include:**
   - A description of the vulnerability.
   - Steps to reproduce the issue (proof-of-concept code is highly appreciated).
   - The potential impact of the exploit.

### Our Response Process

* **Acknowledgment:** You will receive an acknowledgment of your report within 48 hours.
* **Investigation:** We will investigate the issue and may contact you for further information.
* **Resolution:** If confirmed, we will work on a fix. We aim to release a patch within 30 days of the initial report.
* **Disclosure:** Once a fix is released, we will coordinate a public disclosure with credit to you for the discovery (unless you prefer to remain anonymous).

## Preferred Communication

Please encrypt your message if possible or use a secure communication channel. We value your efforts to keep the **TurboLightDI** community safe!
*Thank you for helping keep this project secure.*
